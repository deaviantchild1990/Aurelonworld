WHAT IS THIS

The Descent is a free text adventure set in the world of Aurelon — a sampler of The Breaking, the first novel cycle.

It has four named endings. The first ten readers of each ending preread the unpublished cycle. The first reader of each ending names a character in the books.

This page is where claims are made. The cycle keeps track. The author reads every submission.
